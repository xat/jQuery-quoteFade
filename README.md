# jQuery Quote Fade

jQuery Plugin that lets you simply loop through a bunch of quotes with a fade-effect.

### Usage

    // JS
    $('.quotes').quoteFade({
    	'timer': 5000, // Display time
		'animationTime': 500 // Fade-Animation Time
    });

    //HTML
    <ul class=".quotes">
    	<li>Quote 1</li>
    	<li>Quote 2</li>
    	<li>Quote 3</li>
    </ul>
