# jQuery simpleFade

jQuery Plugin that lets you simply loop through a bunch of items with a fade-effect.

### Usage

´´´´js
$('.quotes').simpleFade({
    'timer': 5000, // Display time
    'animationTime': 500 // Fade-Animation Time
});
´´´´

´´´´html
<ul class="quotes">
    <li>Quote 1</li>
    <li>Quote 2</li>
    <li>Quote 3</li>
</ul>
´´´´
